about=This plugin provides direct access to the Real Estate Information Library API
    provided by Japan's Ministry of Land, Infrastructure, Transport and Tourism (MLIT).

    Features:
    - Retrieve real estate transaction prices and land values
    - Access urban planning data (zoning, fire prevention areas, etc.)
    - Get facility information (schools, hospitals, welfare facilities)
    - View disaster risk data (flood, landslide, tsunami)
    - Export data as GeoPackage, GeoJSON, Shapefile, or CSV

    Requirements:
    - API key from MLIT (free registration required)
    - Internet connection
